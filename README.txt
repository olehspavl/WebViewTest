For correct work of existing pom.xml dependencies in mavenTest project install next plugin:
1. Download *.zip at: https://github.com/simpligility/maven-android-sdk-deployer
2. Unpack it in any folder
3. Run next command from unpacked project folder: mvn install

About files/folders in repo:
- mavenTest - success in apk creating, but this apk doesn't work correctly on device.
- mavenTest.apk - apk generated by mavenTest project. doesn't work correctly on device.
- studioProject - working project for Android Studio.
- studioProject.apk - working apk. it generated by studioProject with the same *.java files and AndroidManifest as mavenTest project.
