For correct work of existing pom.xml dependencies install next plugin:
1. Download *.zip at: https://github.com/simpligility/maven-android-sdk-deployer
2. Unpack it
3. Run next command from unpacked project folder: mvn install

About files/folders in repo:
- crashedTest - old bad attempt to use maven
- apkProblemTest - better attempt. success in apk creating, but this apk doesn't work on device.
- apkProblemTest.apk - apk generated by apkProblemTest project. doesn't work on device.
- goodApk.apk - example of same working apk. it generated by Android studio project with same *.java files and AndroidManifest.
